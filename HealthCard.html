SCENARIO RATIONALE (how targets are chosen)
--------------------------------------------------------------------------------------------
These scenario adjustments do NOT apply a fixed % reduction.
For each stressor, the script selects a target value from the PRIORITY reference population
(MDE8 2140501). The target is typically the LOWER QUARTILE (25th percentile) for 'lower-is-better'
stressors (e.g., nutrients, turbidity), or the UPPER QUARTILE (75th percentile) for 'higher-is-better'
variables (e.g., bank stability, riparian width).

DIRECTIONAL SAFETY:
  - pct=75 variables are increase-only (never recommend decreasing FOREST, RIP_WID, SHADING, etc.).
  - pct=25 variables are decrease-only (never recommend increasing IMPSURF/ISC, nutrients, turbidity, etc.).

ION SAFETY RULE: For ionic/salt indicators (e.g., conductivity, chloride, sulfate), the script NEVER
recommends an increase. Targets use a low-salt benchmark: the 10th percentile (or an optional floor),
and are additionally clamped so proposed ≤ current.

Interpretation: A reported ΔBIBI (e.g., +0.021) means that moving the specified variables to those
regional percentile targets increased the PRIORITY model’s predicted BIBI by 0.021 points.
